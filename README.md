# SnapBack
Use Git to take a snapshot and restore. (for macOS)

## Features
- **START:** Initialize Git
- **SNAP:** Create a Snapshot(Commit)
- **BACK:** Restore Files
- **SHOT:** Restore Files (in **BACK**)
- **SNAP(on BACK):** Create a Snapshot to Confirm Change
